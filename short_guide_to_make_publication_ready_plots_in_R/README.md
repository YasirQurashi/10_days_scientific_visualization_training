# Scientific Visualization in R: 5 Publication-Ready Plots

A concise guide to creating **publication-ready scientific figures** in R using a structured approach.

## 📌 Overview

This project demonstrates how to create **5 essential scientific plots** using the Palmer Penguins dataset.

The focus is on:
- Choosing the right plot
- Improving interpretability
- Making figures publication-ready

## 🎯 Objectives

- Build visualizations from basic → advanced  
- Apply scientific styling  
- Add statistical annotations  
- Use colorblind-friendly palettes  
- Export high-quality figures  

## 📈 Visualization Workflow

Each plot follows this structure:

1. Basic plot  
2. Improved aesthetics  
3. Publication-ready version  

## 📊 Plots Included

### 1. Histogram
**Purpose:** Distribution analysis  

### 2. Boxplot  
**Purpose:** Outliers & group comparison  

### 3. Violin Plot  
**Purpose:** Distribution shape  

### 4. Bar Plot (Mean ± CI)  
**Purpose:** Summary statistics  

### 5. Scatter Plot  
**Purpose:** Relationship analysis  

### 6. Regression Plot  
**Purpose:** Trend and modeling 

## 🧪 Statistical Features

- ANOVA  
- Linear regression  
- Confidence intervals  
- P-value annotations  

## 🎨 Styling & Themes

- `RColorBrewer` for categorical palettes  
- `viridis` for colorblind-friendly colors  
- `ggpubr` for publication-ready themes  

## 🚀 Reproducibility

1. Open `short_guide_to_make_publication_ready_plots_in_R.md`  
2. Run all chunks  
3. All plots will be generated  

## 📤 Output

- 5 publication-ready plots  
- Multi-panel figures  
- High-resolution TIFF exports  

## ⚠️ Common Mistakes

- Using bar plots without error bars  
- Ignoring statistical annotations  
- Poor color choices  
- Overplotting  

## 💡 Key Learnings

- The right plot improves interpretation  
- Visuals must align with statistical analysis  
- Clean design enhances scientific communication
